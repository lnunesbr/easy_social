
-- SUMMARY --

I have developed this project, since there is no other module simple enough to aggregate Twitter, Facebook and Google Plus share buttons in a unique place. Yes, there are a bunch of other social modules, but this one fits perfectly in my needs.


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Go to /admin/settings/easysocial and change it the way you want.
